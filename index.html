← 左の 青いトグル をクリックして ON/OFF。 ON にすると、英語の地震・災害・ニュースのタイトルを Claude Haiku 4.5 で日本語に翻訳します。 翻訳結果は localStorage にキャッシュされ、同じイベントは再翻訳されません。 ※ Anthropic API キーが必要 (未設定なら英語のまま表示)。

← 左の 青いトグル で ON/OFF。
ON にすると 30分に1回、現在の5銘柄価格 + 直近の地震/災害/ニュースを Claude Haiku 4.5 に渡し、 「今後10分の市場ムード補正係数 (0.99〜1.01)」を取得して予測値に反映します。 線形回帰だけでは捉えられない「材料イベント」 (例: 大地震直後の円安一時停止) を予測に組み込む目的。 ※ Anthropic API キーが必要。OFF にすると線形回帰 + 誤差バイアスのみで予測。
「今すぐ実行」を押すと、30分待たずに即座に1回ムード補正を行います。

← 左の 青いトグル で ON/OFF。
ON にすると、世界中を 飛行中の航空機 (最大600機) の現在位置を OpenSky Network から 1分ごとに取得して地球儀/地図上に小さな ✈ マーカーで表示します。マーカーは進行方向を向き、 高度に応じて色が変化 (低=濃い水色、高=明るい水色)。クリックでコールサイン・国・高度・速度を表示。 ※ API キー不要。タイムラインには出ません (常時動いているため)。OFF にすると全マーカー消去。

← 左の 青いトグル で ON/OFF。
(A) 短期軌跡 (全機): 過去最大12点 (約12分) を水色の点線で表示。
(B) 完全航路 (クリック時): 地球儀/地図上の ✈ をクリックすると、 OpenSky から 離陸から現在までの完全な航路 を取得し、金色の太線で描画。 出発空港 (ICAO コード) も「選択中の機体」カードに表示されます。 到着空港は OpenSky の制約上、着陸後にしか判明しません。

← 左の 青いトグル で ON/OFF。
通常は予測してから 10分後 に実値と照合して精度を測ります。 このモードを ON にすると、新しい予測は 10秒後 に検証されます。 「サンプル予測+即検証」を押すと、その場で 5銘柄取得→予測→10秒後検証→精度ダッシュボード更新まで一気に走ります (高速検証モードは自動 ON)。
※ 本番運用では OFF (バイアス補正が短期ノイズに引きずられるため)。動作確認用。

💡 データソースについて: 原油/金/S&P500 はリアルタイム (Yahoo Finance / CoinGecko) を最優先、 取得失敗時のみ Stooq (15-20分遅延) にフォールバックします。CORS 非対応のソースは内蔵プロキシで自動経由。設定は不要です。
